# RAG Document Chatbot 🤖

A Retrieval-Augmented Generation (RAG) based chatbot built using Python, Streamlit, LangChain, FAISS, and Sentence Transformers.

## 🚀 Features
- Upload documents (PDF/Text)
- Semantic search using embeddings
- Context-aware answers using LLM
- FAISS vector database for fast retrieval
- Streamlit UI for interaction

## 🧠 Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Sentence Transformers
- (Optional) Azure OpenAI / OpenAI API
